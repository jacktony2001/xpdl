# xpdl — اسکرپر ویدیو با فشرده‌سازی و ارسال به تلگرام

ابزاری برای استخراج لینک‌های ویدیو از سایت هدف، دانلود، فشرده‌سازی
(با ffmpeg زیر ۴۸ مگ) و ارسال به تلگرام از طریق ربات.

## نصب

```bash
pip install -r requirements.txt
sudo apt install -y chromium-browser ffmpeg   # ffmpeg سیستمی لازمه
cp .env.example .env                          # و مقادیر رو پر کن
```

متغیرهای `.env`:
- `BOT_TOKEN` — توکن ربات تلگرام
- `CHAT_ID` — آیدی چت مقصد
- `WEBSITE_URL` — آدرس صفحه اصلی سایت (پیش‌فرض example.com)

## اجرا

```bash
python main.py
```

## رفع باگ نسخه قبلی (v2)
- حلقه retry فشرده‌سازی اصلاح شد: فایل ورودی دانلود (`temp_video.mp4`)
  تا پایان نگه داشته می‌شد و در هر تلاش از همون encode مجدد انجام می‌گرفت،
  نه از خروجی قبلی (که باعث شکست خاموش می‌شد).
- dedup در `main.py` حالا از آدرس صفحه ویدیو (`page_url`) استفاده می‌کند
  نه اول ۱۰۰ کاراکتر مسیر فایل محلی.

## نکات
- ویدیوهای بالای ۴۸ مگ سعی می‌کنن با کاهش تدریجی کیفیت زیر حد برسن؛
  اگر باز هم نرسیدن، بصورت لینک فرستاده می‌شن.
- دیتابیس SQLite (`sent_content.db`) جلوگیری از ارسال تکراری را انجام می‌دهد.
